Wiimote Whiteboard Help

Support

You can go to the Wiimote Whiteboard website to post your comments, bug reports, feature requests, you name it.

There is also some discussion going on at the wiimoteproject.com forums, especially this thread.

Bluetooth-related problems can probably be solved at the WiiRemoteJ forum.

How you can help

You can support this project by

  • translating the user interface (contact Uwe).
  • improving or translating the documentation (the password can be found at the bottom of each page).
  • donating money to show your appreciation.
